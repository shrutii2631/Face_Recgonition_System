# AI-Powered Face Recognition Attendance System

A full-stack AI-powered web application with complete DevOps pipeline.

## Tech Stack
- **App:** Python, Flask, OpenCV (LBPH), Pandas, Bootstrap
- **DevOps:** Docker, Docker Hub, Jenkins, Terraform, Kubernetes, Prometheus, Grafana

## DevOps Pipeline
1. Docker — Containerisation
2. Docker Hub — Image registry
3. Jenkins — CI/CD automation
4. Terraform — Infrastructure as Code
5. Kubernetes (Minikube) — Container orchestration
6. Prometheus — Metrics monitoring
7. Grafana — Dashboard visualisation

## Quick Start
```bash
# Clone karo
git clone https://github.com/aniketrai07/Face-Recognition-Attendance-System.git

# Docker se chalao
docker pull aniketrai07/face-attendance:v1
docker run -p 5000:5000 aniketrai07/face-attendance:v1
```
